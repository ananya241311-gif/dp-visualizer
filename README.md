# Dynamic Programming Visualizer

## Overview
A simple tool to visualize Dynamic Programming techniques (recursion, memoization, and tabulation) using the Fibonacci sequence.

## Features
- Recursive solution
- Memoized solution (Top-down DP)
- Tabulation approach (Bottom-up DP)
- Comparison of function calls

## Concepts Used
- Recursion
- Dynamic Programming
- Time optimization

## How to Run
1. Clone the repository
2. Run: python dp_visualizer.py

## Example
Input: 5  
Output: Fibonacci value along with function call comparison

## Why I Built This
Built to understand and visualize how Dynamic Programming reduces redundant computations.
